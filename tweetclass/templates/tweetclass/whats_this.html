What's this?

Sentiment Analysis on Twitter for Spanish is a web app developed by Javier Selva for the Universitat Politècnica de València as a Final Degree Project.

It is based on the Lluis Felip Hurtado Oliver and Ferran Pla's Sentiment Analysis in Twitter for Spanish which results in a six-class polarity clasifier. These clases are: Very Positive (P+), Positive (P), Neutral (NEU), Negative (N), Very Negative (N+) and None (NONE). It has an acuracy of 62.88%.

This web app also has a summary functionality which chooses a set of tweets for the given query that represent the main ideas being expressed in the obtained tweets. Furthermore, it also has an Historic section, where past results of a given query are shown in order to compare its polarity evolution.
